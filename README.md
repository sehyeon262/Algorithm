## ✔️ 파일 네이밍 규칙
```
난이도_문제번호_문제이름.확장자
```

#### 예시
```
골드_15989_123더하기4.py
골드_15989_123더하기4.java
D3_24396_공과상자.py
D3_24396_공과상자.java
```

- 난이도: `브론즈 / 실버 / 골드 / 플래티넘 / D1~D6`
- 문제 이름은 가독성을 위해 한글 사용

## ✔️ SWEA 학습 폴더 규칙

#### SWEA는 강의/학습 단위별 폴더로 관리합니다.
```
0813_Stack1/
0814_Stack2/
0820_Queue/
0904_Recursive&Bruteforce/
```

- 날짜 + 주제 기반
- 폴더 안에 여러 문제 풀이 파일 포함
